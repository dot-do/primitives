# AI Site

Framework for building AI-powered websites.

## Features

- AI-first website framework
- Integration with AI primitives
- Responsive design components
- SEO optimizations

## Installation

```bash
pnpm add ai-site
# or
npm install ai-site
# or
yarn add ai-site
```

## Usage

```typescript
import { createSite } from 'ai-site'

const site = createSite({
  name: 'My AI Site',
  theme: 'light',
  plugins: ['ai-chat', 'ai-search']
})

export default site
```
