# Hero Component Example

<Hero 
  title="Innovative Software Solutions"
  subtitle="Discover our collection of powerful tools and frameworks designed to streamline your development process"
  ctaText="Explore Now"
  ctaLink="#categories"
/>

## Usage

The Hero component accepts the following props:

- `title` - The main heading text
- `subtitle` - Supporting text below the heading
- `ctaText` - Text for the call-to-action button
- `ctaLink` - URL for the call-to-action button

## Examples

### Default Hero

```jsx
<Hero />
```

### Custom Hero

```jsx
<Hero 
  title="Welcome to Our Platform"
  subtitle="Powerful tools to help you build amazing experiences"
  ctaText="Get Started"
  ctaLink="/signup"
/>
```
